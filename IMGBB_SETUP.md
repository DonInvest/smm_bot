# Настройка Imgbb для загрузки изображений в Farcaster

## Что такое Imgbb?

Imgbb — это бесплатный сервис для хостинга изображений, который мы используем как альтернативу Imgur для загрузки фото в посты Farcaster.

## Как получить API ключ

1. Перейдите на https://api.imgbb.com
2. Прокрутите вниз до раздела "API Key"
3. Нажмите "Get API Key" или просто скопируйте ключ, если он уже показан
4. Добавьте ключ в ваш `.env` файл:
   ```
   IMGBB_API_KEY=ваш_ключ_здесь
   ```

**Важно:** Регистрация не требуется! Просто получите ключ и используйте его.

## Лимиты

- Бесплатный тариф: до 32 МБ на изображение
- Без ограничений по количеству загрузок
- Изображения хранятся неограниченно долго

## Альтернативы

Если Imgbb не подходит, можно использовать другие сервисы:
- **UploadMe** (https://uploadme.me) — также бесплатный
- **Cloudinary** (https://cloudinary.com) — более продвинутый, но требует регистрации

Для замены сервиса нужно изменить функцию `upload_image_to_imgbb()` в `main.py`.
